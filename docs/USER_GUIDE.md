## Установка

### Chrome

```bash
git clone https://github.com/dorofeevb1/transcript.git
cd transcript
make install-extension
make build-extension
```

1. `chrome://extensions` → **Режим разработчика**
2. **Загрузить распакованное** → папка `extension/dist`
3. Закрепите иконку на панели (по желанию)

После обновления кода: **↻ Обновить** на карточке расширения.

### Firefox 142+

```bash
make build-firefox-zip
```

`about:debugging` → **Загрузить временное дополнение** → `extension/dist-firefox/manifest.json`.

Постоянная установка — через [AMO](PUBLISHING_FIREFOX.md) после одобрения версии.

### Требования

| | Chrome | Firefox |
|---|--------|---------|
| Браузер | Chrome / Chromium MV3 | Firefox **142+** (десктоп) |
| Субтитры | ✅ | ✅ |
| Из аудио (Whisper) | ✅ | ❌ |
| Интернет | для платформ и Google Translate | то же |

## Быстрый старт

1. Откройте страницу с видео (YouTube / Rutube / VK).
2. Нажмите иконку **«Стенограмма»**.
3. Режим **«Субтитры видео»** → **«Получить текст»**.

Сервер **не нужен**, если у ролика есть субтитры.

---

## Режимы получения текста

| Режим | Что делает | Скорость | Сервер? |
|--------|------------|----------|---------|
| **Субтитры видео** | Готовые субтитры с сайта | Секунды | Нет |
| **Из аудио (Whisper)** | Распознавание речи | Минуты | Да (Chrome) |

### Субтитры видео (рекомендуется)

- YouTube, Rutube, VK — если есть дорожка субтитров (в т.ч. автогенерация).
- **Не** распознаёт голос — только текст от плеера.
- Rutube/VK: включите CC, подождите 5–10 с, затем «Получить текст».

### Из аудио (только Chrome)

- Когда субтитров нет.
- Запущен **whisper-server** на `http://127.0.0.1:8765`.
- **Play** и **звук** на вкладке; лимит записи — в настройках (по умолчанию 5 мин).

## Интерфейс popup

### История

- Недавние стенограммы в браузере.
- **Очистить** — с подтверждением в окне расширения.

### Отображение

| Режим | Вид |
|--------|-----|
| **По минутам** | `[00:00 – 01:00] текст…` |
| **По фразам** | `[04:08] реплика…` |

### Прогресс

Перевод и Whisper идут **в фоне**. Можно закрыть popup — при открытии снова виден прогресс и результат.

Вкладку с видео для Google Translate лучше не закрывать.

### Отмена

**«Отмена»** — останавливает запись/STT (Chrome).

## Перевод

1. Получите стенограмму.
2. **Перевод** → язык → **«Применить»**.
3. Пустой язык — оригинал.

Длинные ролики (>200 фраз) — **по минутным блокам**.

**Источники** (по приоритету):

1. **Argos** на локальном сервере (офлайн после установки моделей).
2. **Google Translate** (через вкладку или service worker).

## Локальный сервер (Whisper)

```bash
cd whisper-server
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

→ **http://127.0.0.1:8765** · проверка в настройках расширения (⚙).

**Argos (перевод офлайн):**

```bash
pip install argostranslate
python install_translate_models.py
```

Или: `make install-translate` из корня репозитория.

Подробнее: **[WHISPER-SERVER.md](WHISPER-SERVER.md)**

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| `WHISPER_MODEL` | `small` | `base`, `small`, `medium` |
| `WHISPER_DEVICE` | `auto` | `cpu`, `cuda`, `auto` |
| `TRANSLATE_ENGINE` | `auto` | `auto`, `argos`, `google` |

**Docker:** `cd whisper-server && docker compose up --build`

## Частые вопросы

**Нужен ли сервер для YouTube с субтитрами?**  
Нет.

**Чем STT отличается от «Субтитры»?**  
STT = «Из аудио», распознавание. «Субтитры» = готовый текст с сайта.

**Почему перевод долгий на 5+ часах?**  
Много минутных блоков; повтор того же языка — из кэша.

**Offline?**  
Субтитры — нужен интернет к платформе. Whisper + Argos — после загрузки моделей можно без сети.

**Прямые трансляции?**  
Ограниченно; STT для live не рекомендуется.

**Firefox — «Из аудио»?**  
В Firefox этот режим **недоступен**; используйте субтитры.

## Решение проблем

| Симптом | Действие |
|---------|----------|
| «Нет субтитров» | CC в плеере, F5, 5–10 с; или Whisper (Chrome) |
| «Перевод недоступен» | Вкладка с видео, F5; whisper-server; подождать 1–2 мин |
| `Receiving end does not exist` | ↻ расширение → F5 на видео |
| «Сервер недоступен» | `python main.py`, URL `http://127.0.0.1:8765` |
| Popup не видит видео | Активная вкладка = страница ролика |
| «Из аудио» зависает | Play + звук; лимит в настройках |
| Дубли (VK) | Обновите расширение, получите текст заново |

## Настройки

- **URL сервера** — whisper-server
- **Язык субтитров** — ru / en / авто
- **Язык распознавания** — для Whisper
- **Макс. запись «Из аудио»** — 3–60 мин
- **Размер блока** — 30 / 60 / 120 с
- **Помечать тишину** — `[тишина]` в пустых минутах
- **Язык интерфейса** — ru, en, uk, de, es, fr
