# Store assets — youtube-transcript-extension

Generated by FRONTEND-AGENT in Phase B. Intended for Chrome Web Store and
Firefox AMO submission.

```
store-assets/
├── icons/          # source SVG + rendered PNGs at 16/32/48/128
├── promo/          # 440×280 promo tile (EN/RU) + 1400×560 marquee (EN)
└── screenshots/    # 5 shots × 2 locales (EN/RU) at 1280×800
```

## Status

- **Icons (16/32/48/128):** real — rendered from `icons/icon-source.svg` at
  full density and pixel-snapped at small sizes via `icons/icon-small.svg`.
  Same files copied into `extension/icons/` so the live extension uses them.
- **Promo tile 440×280:** real — both EN and RU rendered from SVG.
- **Marquee 1400×560:** real (EN only) — optional, only needed for Featured
  submission.
- **Screenshots 1280×800:** mockups. Layouts and copy are correct, but
  these are SVG renders, not live browser captures. QA-AGENT regenerates
  from the real popup in Phase C per `screenshots/README.md`.
