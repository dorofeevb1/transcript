# Chrome Web Store — листинг RU

Важно для RU-рынка: Rutube и VK Видео упоминаем сразу в названии и в первой строке описания. Это даёт дифференциацию от десятков «YouTube transcript» расширений, в которых русскоязычные платформы вообще не поддерживаются.

## Название (≤45 символов)

**Стенограмма видео: YouTube, Rutube, VK**

(40 символов. Бренд «Стенограмма видео» совпадает с тем, как пользователи и так гуглят. Три платформы — основной USP.)

Запасные варианты:
- `Расшифровка видео — YouTube, Rutube, VK` (39)
- `Субтитры и стенограмма: YouTube/Rutube/VK` (41)

## Краткое описание (≤132 символа)

**Полный текст видео с YouTube, Rutube и VK. Экспорт в TXT/SRT/JSON, перевод, офлайн-распознавание Whisper.**

(105 символов. Три платформы, четыре фичи — без воды.)

## Полное описание (≤1500 символов)

Получите полный текст любого видео с YouTube, Rutube или VK — бесплатно, без копирования с экрана.

Расширение читает дорожку субтитров (ручную или автоматическую) и показывает текст по минутам или по фразам. Экспорт в TXT, SRT и JSON. Перевод одним нажатием. Дополнительно: если у ролика субтитров нет, есть локальный Whisper-сервер, который распознаёт речь прямо на вашем компьютере.

Возможности:

- Один popup для YouTube, Rutube и VK Видео
- Два режима отображения — по минутам и по фразам
- Копирование в буфер, экспорт в TXT, SRT, JSON
- Перевод через Google Translate или офлайн через Argos
- 6 языков интерфейса — русский, английский, украинский, немецкий, испанский, французский
- Светлая, тёмная и системная темы
- Локальная история и кэш — повторное открытие мгновенное

Опционально — офлайн-распознавание речи:

- Whisper-сервер на FastAPI запускаете у себя
- Argos для перевода без интернета (после загрузки моделей)
- Аудио никогда не уходит с вашего компьютера

Полезно студентам, журналистам, исследователям, авторам контента, тем, кто учит языки, и всем, кому нужен поисковый текст того, что было сказано.

Приватность: никаких серверов автора, никакой аналитики, никакого remote code. Стенограммы и настройки лежат только в локальном хранилище браузера. Google Translate срабатывает только когда вы нажали «Применить» в блоке перевода. Whisper STT работает полностью на вашей машине через loopback.

Исходный код открыт. Поддержка и баг-репорты — через issues в репозитории на GitHub.

## Обоснования разрешений (для CWS-ревью)

**storage** — настройки (язык интерфейса, режим отображения, URL локального Whisper-сервера), плюс локальный кэш недавних стенограмм, чтобы то же видео открывалось мгновенно. Только локальное хранилище.

**scripting** — инжектит маленькие статические функции в открытую вкладку с видео, чтобы прочитать публичные метаданные плеера: Innertube-ответ YouTube, `play/options` Rutube, состояние страницы VK. Функции уже собраны в bundle; пользовательский ввод никогда не превращается в код.

**activeTab** — нужен, чтобы popup знал, какое именно видео сейчас открыто. Другие вкладки не читаются.

**tabCapture** — нужен только для режима «Из аудио». Когда у ролика нет субтитров и пользователь сам запустил локальный Whisper-сервер, расширение записывает звук текущей вкладки и шлёт его на `http://127.0.0.1:8765/transcribe`. Запись никогда не покидает машину пользователя. В сборке Firefox этого разрешения нет.

**offscreen** — Chrome не разрешает `tabCapture` напрямую из service worker, нужен document-контекст. Создаётся один короткоживущий offscreen-документ (`USER_MEDIA`), он держит поток ровно столько, сколько идёт запись, и закрывается.

**host: youtube.com, youtu.be** — загрузка дорожки субтитров и Innertube-ответа для текущего видео.

**host: rutube.ru** — субтитры Rutube отдаются публичным JSON-эндпоинтом `play/options`. Аккаунт Rutube не требуется.

**host: vk.com, vkvideo.ru, okcdn.ru, vkuser.net, mycdn.me** — плеер VK отдаёт субтитры и медиа с этих доменов. MAIN-world-скрипт только слушает URL запроса субтитров (никаких модификаций), а основной код потом скачивает уже известный URL.

**host: translate.googleapis.com** — опциональный перевод. Срабатывает только когда пользователь нажал «Применить». По умолчанию выключен.

**host: http://127.0.0.1:8765, http://localhost:8765** — loopback к локальному Whisper-серверу. Трафик не покидает компьютер.

**Remote code:** отсутствует. CSP — `script-src 'self'; object-src 'self';`.
