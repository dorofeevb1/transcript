## RU market specifics

The Rutube + VK angle is the moat. Almost every English transcript extension ignores Russian platforms. Make sure those keywords appear in:

- Title (`Стенограмма видео: YouTube, Rutube, VK`)
- Short description (first sentence)
- First two bullets of the full description
- Screenshot caption #1

## Action items

- P0 terms — title or short description.
- P1 terms — full description body, screenshot captions, Habr/vc.ru tags.
- P2/P3 — sprinkle in feature bullets and launch posts. Do not stuff.
- Skip "rewrite as keywords" — CWS algorithm penalizes obvious stuffing.
